How to Setup Universe


ignoreUniverse

Ignore universe is a small script used to define localStorage keys that you need to be hidden from Universe's filesystem. You must include this in your website / program.

let currentContent; let newContent = []; let hidContent = [ // Put your localStorage keys that you want to be hidden from the Universe filesystem here. ]; function startLoad() { currentContent = JSON.parse(localStorage.getItem('appFiles')) || []; currentContent.forEach(function(added) { newContent.push(added); }); hidContent.forEach(function(added2) { newContent.push(added2); }); localStorage.setItem('appFiles', JSON.stringify(newContent)); } // Please call this in your html document.
